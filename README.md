# CIRS
CIRS (Campus Incident Reporting System) is a full-stack web platform that enables students and staff to securely report campus incidents, track report status, and access emergency support. Built with React, Tailwind CSS, Node.js, Express, PostgreSQL, and AI/NLP, the system promotes transparency and campus safety through smart automation and data-driven insights.

Anonymous or named incident reporting
File and media upload support
Real-time status tracking (New → Investigating → Resolved)
Admin dashboard with analytics (Chart.js)
AI-powered auto-categorization and anomaly detection
Secure JWT authentication and role-based access

Frontend: React, Tailwind CSS
Backend: Node.js, Express.js
Database: PostgreSQL (Supabase)
AI/ML: Python (spaCy, Flask/FastAPI)
Hosting: Render / Vercel
